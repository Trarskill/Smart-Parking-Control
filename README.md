# Smart Parking Control

## Опис проєкту

Це навчальний веб-проєкт для імітації системи розумного контролю паркінгу. Інтерфейс включає:

- сторінку статистики (`index.html`),
- інтерактивну мапу паркінгу (`map.html`),
- сторінку опису системи (`about.html`).

Проєкт створено в рамках курсу "Web-програмування та дизайн". Дизайн натхненний макетом з Figma: https://www.figma.com/design/ViNgenBFTJ322h0lrmVoik/Smart-Parking?node-id=0-1&t=F0T8Q61qjFeOQMw5-1

## Основні можливості

- Динамічна симуляція зайнятості паркомісць
- Віджет з показниками: вільні місця, відсоток заповненості, середній час паркування
- Журнал останніх подій з генерованими автомобільними номерами
- Інтерактивна мапа, де можна клікати по сектору для зміни статусу
- Налаштування через модальне вікно: кількість місць та швидкість симуляції
- Збереження стану в `localStorage`

## Технології

- HTML
- CSS + Tailwind CSS
- JavaScript
- Локальне сховище браузера (`localStorage`)

## Файлова структура

- `index.html` — сторінка статистики паркінгу
- `map.html` — інтерактивна карта секторів паркінгу
- `about.html` — інформаційна сторінка про систему
- `tailwind.config.js` — налаштування кольорів та шрифтів Tailwind
- `js/script.js` — основна логіка симуляції та рендерингу
- `js/mapHandlers.js` — обробка карти й кліків по місцях
- `js/settings.js` — робота з модальним вікном налаштувань
- `js/randomUtils.js` — допоміжні утиліти для випадкової генерації
- `assets/` — шрифти, іконки та зображення

## Запуск проекту

1. Відкрийте папку проекту в браузері або локальному сервері.
2. Відкрийте `index.html` для перегляду дашборда статистики.
3. Відкрийте `map.html` для перегляду інтерактивної мапи.
4. Використайте кнопку "Settings" для налаштування кількості місць та швидкості симуляції.

> Зауваження: для правильної роботи Tailwind через CDN достатньо просто відкрити файли у браузері. Якщо потрібно, можна запустити простий локальний сервер через VS Code Live Server або інший HTTP-сервер.
